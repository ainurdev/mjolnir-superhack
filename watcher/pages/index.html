To start streaming on Mjolnir, follow these steps:

  1. Install OBS
    1. Start OBS
    2. Go to "Settings" > "Output"
    3. Change "Output Mode" to "Advanced"
    4. Go to the "Recording" tab
    5. Change "Type" to "Custom Output (FFmpeg)"
    6. Change "Container Format" to "hls"
    7. Select an empty directory for "File path or URL", Remember this directory
    8. Click "OK" to save and exit settings
  2. Start recording your screen
  3. Select the recording directory you chose in step 1.7
  4. Click the Upload button
